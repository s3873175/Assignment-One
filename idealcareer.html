IDEAL CAREER

FRONT END DEVELOPER

A front-end developer is responsible for the visual components of a website or software application that consumers can interact with. They typically work alongside back-end developers who construct the servers that allow the platform to operate on the internet. Front-end developers additionally collaborate with graphic designers who provide their works of logos, images, video, or anything the client would like presented as a visual for customers to see. I find this area intriguing as I have always had a creative outlet in life that drives me to complete a task, whether that be music or other art forms. With the use of current technology, you can bring a design to life that functions as a webpage or application online. It is very rewarding that an unlimited amount of people across the globe can navigate through a project that your team has spent a great amount of time creating.

Server Environment Manager

Ensures maintaining all forms of data including hardware, applications, websites, cloud and local storage, emails and more. A server environment manager is in charge of guaranteeing data is kept up to date, backed up, and stored securely for their client. They must uphold servers running at optimal performance for others to rely on doing their daily tasks.

API Integration

Allows two software components to talk and perform with one another and present shared information. An excellent example of this is the integration of a weather app. The application cannot acquire weather predictions without sourcing them from a forecast database. API integration connects both the application and database to show the same data, users can then quickly glance at the said data on the application installed, without having to search up the database online.

CMS (Content Management Systems) Develoment

A system that individuals can use to construct a webpage without having the knowledge of an experienced developer. A CMS is an easily operated interface that can publish posts, customise pages, and handle content while avoiding the use of complicated coding or programming skills.
